 # Registro venta en un  tienda #

  Project Description

_This script was created to help store owners keep track of daily sales in a simple and organized way. The program allows the user to register sales, calculate totals, and generate a final summary of the day's activity._

 ## User History ##

As a store administrator,I want to record the sales made during the day using a script,
so that I can obtain a clear summary of the products sold, the quantities, and the total revenue collected at the end of the day.

## Functional Requirements ##

The system must:

Allow the user to register multiple sales during the program execution.

Request the necessary information for each sale (product, price, and quantity).

Automatically calculate the total value of each sale.

Generate a final sales summary that includes:

List of registered products

Total quantity sold per product

Total revenue collected during the day.

## Features ##

Register multiple sales during program execution.

Request product information from the user (product name, price, and quantity).

Automatically calculate the total value of each sale.

Generate a summary of all registered sales.
